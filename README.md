# Deadline Tracker Extension

A simple Chrome extension to track deadlines in a compact popup.

## Features

- Add deadlines with optional time
- See remaining time in days, hours, and minutes
- Color-coded urgency status
- Edit events
- Delete events
- Mark events as done
- Set reminder notifications

## Files

- `manifest.json` — extension configuration
- `popup.html` — popup layout
- `popup.js` — popup logic
- `background.js` — reminder and notification logic

## How to use

1. Download or clone this repository
2. Open `chrome://extensions/`
3. Turn on **Developer mode**
4. Click **Load unpacked**
5. Select this project folder

## Notes

This extension is currently shared as an unpacked Chrome extension for free use.
